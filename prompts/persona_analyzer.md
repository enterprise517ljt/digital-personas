# Persona 分析指南 — L1-L5 五层结构

## 分析原则

1. **来源优先**：从实际语料推断，不从外部标签臆测
2. **行为 > 声明**：他说过什么不重要，他实际怎么做才重要
3. **分层提取**：每个层次独立分析，避免混淆

## L1 — 硬规则（不可违背）

这一层定义角色的绝对底线，触及时角色必须拒绝或强烈反应。

提取维度：
- 对什么话题绝对不妥协？
- 什么行为会被他直接拉黑/强烈反驳？
- 什么错误他完全不能容忍？
- 什么价值判断是"红线"？

关键词：底线、禁忌、绝对不行

## L2 — 身份定位

这一层定义角色是谁，决定他看待问题的基本立场。

提取维度：
- 职业身份（他怎么介绍自己？）
- 社会角色（老板视角？创业者视角？普通人视角？）
- 性别/年龄/阶层带来的天然立场
- 他认为自己在某个话题上的"资格"从何而来

## L3 — 表达风格

这一层决定输出时的语言形式，是角色辨识度的核心。

提取维度：
- 口头禅（高频出现的固定表达）
- 语气（强势/柔和/调侃/严肃）
- 句式习惯（短句/长句/反问/排比）
- 常用比喻/类比风格
- 对网络用语/流行语的态度
- 是否有地域方言特色

## L4 — 决策模式

这一层决定面对问题时，他会怎么判断优先级。

提取维度：
- 面对求助，他的第一反应是什么？
- 他更看重可行性还是理想性？
- 他给建议的逻辑是什么？（先分析原因/先给结论/先问背景）
- 什么时候他会直接说"你错了"而不是委婉表达？
- 什么时候他会选择沉默或转移话题？

## L5 — 人际行为

这一层决定他对不同类型的人如何差异化回应。

提取维度：
- 对新手小白 vs 老手的不同态度
- 对认真求助 vs 伸手党的不同态度
- 对男性/女性的不同态度
- 对杠精/黑粉的态度和处理方式
- 他怎么维护自己的粉丝群体

## 标签翻译参考表

| 用户给出的标签 | 翻译为行为规则 |
|---|---|
| 接地气 | 不用专业术语，用大白话；善用方言/俚语 |
| 专业 | 先讲原理，再给方案；引用数据/案例 |
| 犀利 | 直接指出问题，不绕弯子；常用反问句 |
| 温柔 | 语气柔和，先共情再给建议；较少直接批评 |
| 实战派 | 只讲已验证的方法；不说"理论上" |
| 鸡汤 | 用故事和比喻；偏感性，少给具体步骤 |
