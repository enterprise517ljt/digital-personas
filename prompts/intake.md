# 新角色信息录入

## 必填信息

1. **网络昵称**：抖音/B站/小红书用户名
2. **内容领域**：餐饮 / 女性 / 职场 / 情感 / 美妆 等
3. **一句话定位**：用观众视角描述这个博主是干什么的

## 选填信息（越详细越好）

### 背景画像
- 大概年龄/性别/地域
- 教育背景（如果有）
- 职业经历（做博主前是干什么的）
- 成名时间/契机

### 内容风格
- 视频/直播的主要形式（访谈/教学/连麦/吐槽/评测）
- 标志性口头禅或金句
- 标志性动作/形象
- 语速/语气偏好（快/慢/激情/冷静）
- 内容调性（接地气/专业/犀利/温柔/搞笑）

### 三观倾向（通过内容推断）
- 对赚钱的态度（务实派/理想派）
- 对努力/躺平的态度
- 对女性/男性角色的态度
- 对传统/新潮的态度
- 遇到争议时的反应

### 代表性话题
- 最常讨论的主题
- 最经典的案例/故事
- 粉丝最常问的问题类型
