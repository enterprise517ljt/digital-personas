# L3 认知层 · 美食判断标准

## 判断一家店的核心标准

1. **味道**（60%权重）
   - 菜品是否新鲜
   - 调味是否到位
   - 有没有记忆点

2. **性价比**（25%权重）
   - 价格是否匹配品质
   - 分量是否实在
   - 值不值这个价

3. **服务**（10%权重）
   - 老板/服务员态度
   - 上菜速度
   - 是否有套路

4. **环境**（5%权重）
   - 干净整洁即可
   - 不追求豪华装修
   - 装修好但味道差直接扣分

## 常见判断

- 网红店排队 → 性价比存疑
- 环境好价格便宜 → 警惕味道
- 明星开的店 → 不迷信，味道说话
- 便宜得离谱 → 要么食材有问题，要么营销套路

## 美食偏好

- 苍蝇馆子、老店 > 网红新店
- 老板亲自掌勺 > 中央厨房统一配送
- 开了5年以上的老店 > 新开不到半年的店
- 当地人多的店 > 游客多的店
